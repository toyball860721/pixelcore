# BI 仪表板文档

## 📋 概述

PixelCore BI 仪表板提供强大的数据可视化和报表生成功能，帮助用户实时监控业务指标和生成分析报告。

## 🎯 核心功能

### 1. 实时数据可视化

**支持的图表类型**:
- 📈 折线图 (Line Chart) - 趋势分析
- 📊 柱状图 (Bar Chart) - 对比分析
- 🥧 饼图 (Pie Chart) - 分布分析
- 📉 面积图 (Area Chart) - 累积趋势

**特性**:
- 实时数据更新（10s/30s/1m/5m 可配置）
- 多时间范围选择（1小时/24小时/7天/30天）
- 响应式设计，自适应屏幕大小
- 交互式图表，支持缩放和悬停提示

### 2. 关键指标卡片

**显示指标**:
- 总事件数
- 活跃用户数
- 平均响应时间
- 错误率

**特性**:
- 实时数值更新
- 变化趋势指示（上升/下降）
- 百分比变化显示

### 3. 报表生成器

**报表类型**:
- 摘要报表 (Summary Report)
- 详细报表 (Detailed Report)
- 自定义报表 (Custom Report)

**可选指标**:
- Total Events
- Active Users
- Page Views
- Response Time
- Error Rate
- Conversion Rate
- Revenue
- User Engagement

**导出格式**:
- CSV - 逗号分隔值
- Excel - Microsoft Excel 格式
- PDF - 便携式文档格式

**调度功能**:
- 支持 Cron 表达式
- 定时自动生成报表
- 邮件分发（计划中）

### 4. 报表配置管理

**功能**:
- 保存报表配置
- 加载已保存配置
- 快速生成常用报表

## 🚀 使用指南

### 1. 查看实时仪表板

```typescript
import { BIDashboard } from './BIDashboard';

function App() {
  return <BIDashboard />;
}
```

**操作步骤**:
1. 选择时间范围（1H/24H/7D/30D）
2. 配置自动刷新（开启/关闭）
3. 设置刷新间隔（10s/30s/1m/5m）
4. 查看实时图表和指标

### 2. 生成报表

```typescript
import { ReportGenerator } from './ReportGenerator';

function App() {
  return <ReportGenerator />;
}
```

**操作步骤**:
1. 输入报表名称
2. 选择报表类型
3. 勾选需要的指标
4. 选择时间范围
5. 选择导出格式
6. （可选）设置调度表达式
7. 点击"Generate Report"生成报表

### 3. 保存和加载配置

**保存配置**:
1. 配置好报表参数
2. 点击"Save Configuration"
3. 配置将保存到列表

**加载配置**:
1. 在"Saved Report Configurations"中找到配置
2. 点击"Load"按钮
3. 配置将自动填充到表单
