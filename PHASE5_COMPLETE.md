## 🎨 核心功能展示

### 1. AI 推荐系统

```rust
// 初始化推荐引擎
let mut engine = RecommendationEngine::new("redis://127.0.0.1:6379").await?;

// 训练模型
engine.train(interactions).await?;

// 获取推荐
let request = RecommendationRequest {
    user_id: user_id,
    limit: 10,
    item_type: None,
    exclude_items: None,
};
let response = engine.recommend(request).await?;
```

### 2. AI 增强搜索

```rust
// 初始化搜索引擎
let engine = SearchEngine::new(config).await?;

// 索引文档
engine.index_documents(documents).await?;

// 搜索
let query = SearchQuery {
    query: "rust programming".to_string(),
    limit: 10,
    ..Default::default()
};
let response = engine.search(query).await?;

// 自动完成
let suggestions = engine.autocomplete("rus", 5);
```

### 3. 数据仓库

```rust
// 初始化数据仓库
let warehouse = DataWarehouse::new(config).await?;

// 查询事件
let events = warehouse.query_events(
    Some("page_view"), None, None, None, 10
).await?;
```
